文心一言是百度推出的大语言模型产品，基于文心大模型（ERNIE）系列，面向中文用户提供对话、创作、分析等能力。文心一言 4.0 在理解、生成、推理、多模态等方面均有显著提升，尤其在中文场景下的语义理解、文化适配、专业领域表达上具有优势。文心一言不仅提供通用对话，还通过插件、API、企业版等形式支持智能体化应用，是国产大模型生态中的重要一员。

**文心一言的智能体能力**

文心一言支持插件生态，用户可启用搜索、文档解析、计算、天气、翻译等插件，通过自然语言触发工具调用。文心一言具备多模态能力，可理解图片内容、生成图像（文心一格），适用于图文混合的智能体场景。文心一言的 API 开放 Function Calling 能力，开发者可自定义工具并接入业务系统，构建客服、知识问答、数据分析等智能体应用。文心一言在企业市场的布局包括私有化部署、数据安全合规、行业定制等，满足政企客户对可控、可审计智能体的需求。
