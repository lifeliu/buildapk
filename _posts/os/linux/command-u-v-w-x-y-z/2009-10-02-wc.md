**wc**（Word Count）统计指定文件中的字节数、字数、行数，并将统计结果显示输出。该命令统计指定文件中的字节数、字数、行数。如果没有给出文件名，则从标准输入读取，wc同时也给出所指定文件的总统计数。

**命令**
```
wc [OPTION]... [FILE]...
wc [OPTION]... --files0-from=F
```

参数选项：
```
-c 统计字节数。
-l 统计行数。
-m 统计字符数。这个标志不能与 -c 标志一起使用。
-w 统计字数。一个字被定义为由空白、跳格或换行字符分隔的字符串。
-L 打印最长行的长度
```
